# Movie Box Office Revenue Analysis

## Project Overview
This project explores the factors that influence global movie revenue using the TMDB 5000 Movie Dataset. Utilizing R, the study covers the full data science lifecycleâ€”from raw data cleaning to exploratory visualization and statistical modeling.

## Key Features
* **Data Preprocessing**: Standardized data formats and addressed missing values within the TMDB dataset.
* **Exploratory Data Analysis (EDA)**: Created visual insights into revenue trends, budget correlations, and genre performance.
* **Statistical Modeling**: Applied regression analysis to identify the most significant predictors of box office success.
* **Detailed Reporting**: Comprehensive findings and data interpretations are documented in the final PDF report.

## Tech Stack
* **Language**: R
* **Libraries**: tidyverse (ggplot2, dplyr), rmarkdown

## File Structure
* **467_project-copy.Rmd**: The primary R Markdown file containing all analysis and visualization code.
* **tmdb_5000_movies.csv**: The source dataset used for this project.
